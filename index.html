$.gallery

Плагин jQuery для реализации адаптивной всплывающей фотогалереи

View on GitHub Download .zip Download .tar.gz

Описание

Адаптивная всплывающая фотогалерея, расширяющая плагин $.popup, со следующими ключевыми особенностями:

  • Адаптивность
  • Максимальная отзывчивость на десктопе и на мобильных устройствах
  • Полноценный swipe на touchscreen-дисплеях
  • Поддержка групп изображений (один вызов $.gallery() для нескольких галерей)
  • Элегантная плавная анимация перелистывания
  • Полный контроль над стилями и разметкой прямо в html
  • Добавление и удаление превью без повторной инициализации плагина
  • Простой и удобный механизм расширения функционала
  • Небольшой размер
